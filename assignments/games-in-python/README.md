# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build a classic Hangman-style word guessing game using Python. Practice string manipulation, loops, conditionals, user input, and game state management while creating a fun interactive experience.

## 📝 Tasks

### 🛠️ Build the Game Logic

#### Description
Create the main Hangman game flow, including word selection, guess processing, and tracking game progress.

#### Requirements
Completed program should:

- Randomly choose a word from a predefined list
- Track correct and incorrect letter guesses
- Reveal letters in the hidden word as they are guessed
- End the game after a win or when attempts are exhausted

### 🛠️ Handle User Interaction

#### Description
Prompt the player for input and display the game state clearly after each guess.

#### Requirements
Completed program should:

- Prompt the user to guess one letter at a time
- Display the current word progress in `_ _ _` format
- Show remaining attempts and letters already guessed
- Provide win and lose messages at the end of the game

### 🛠️ Add Optional Enhancements

#### Description
Add extra polish or replayability to the game with additional features.

#### Requirements
Completed program should:

- Allow the player to play again after a game ends
- Optionally implement difficulty levels or word categories
- Display a simple hangman progress indicator or ASCII art for incorrect guesses
